# 콘텐츠 분할 요소

해당 요소들은 콘텐츠나 레이아웃에 어떤 영향도 주지 않습니다. 그룹핑과 스타일 작업을 위해 사용합니다.

## div

주로 구역을 나누기 위해 사용 됩니다.

```html
<div>
  아무런 영향이 없다
</div>
```

## span

주로 텍스트에 스타일 변경을 위해 사용 됩니다.

```html
<span>
  컨텐츠
</span>
```

### 차이점

div는 dispaly가 block인 요소이고 span은 dispaly가 inline인 요소입니다.
여기서 `dispaly`는 css에서 설명 합니다.
